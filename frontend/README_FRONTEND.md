# Frontend - Customer Experience Agent

Next.js frontend for the Customer Experience Agent with TypeScript and Tailwind CSS.

## Quick Start (Windows PowerShell)

```powershell
cd frontend
npm install
copy .env.example .env.local
npm run dev
```

The app will be available at [http://localhost:3000](http://localhost:3000).

## Prerequisites

- Node.js 18+
- Backend running at http://127.0.0.1:8000

## Configuration

Copy `.env.example` to `.env.local`:

- `NEXT_PUBLIC_API_BASE` - Backend API URL (default: http://127.0.0.1:8000)

## Features

- Customer list with selection
- Customer detail view
- Seed demo data button
- Dark mode support
