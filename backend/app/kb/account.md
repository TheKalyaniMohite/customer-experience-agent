# Account Management

## Profile Settings
- Update name and email in Settings > Profile
- Change password in Settings > Security
- Enable 2FA for additional security

## Team Management
- Invite team members via Settings > Team
- Assign roles: Admin, Manager, Agent
- Set permissions per role

## Billing
- View invoices in Settings > Billing
- Update payment method
- Download receipts
- Cancel or upgrade plan

## Data Export
- Export all customer data as CSV
- Export messages and tickets
- Request full data export (GDPR)

## Account Deletion
To delete your account:
1. Export your data first
2. Go to Settings > Account
3. Click "Delete Account"
4. Confirm via email

Note: Account deletion is permanent and cannot be undone.

## Security Features
- Two-factor authentication (2FA)
- Session management
- Login history
- IP allowlisting (Enterprise)
